🛠 CCT Tool — MCB Programming

Hardware Required

🔌
ACTiSYS USB IrDA Transceiver
Driver: actisys.com/Downloads.html (IR224UN-DriverInstaller). Win10 older adapters: ifamilysoftware.com/news37.html "Windows 64-bit Fix".

Sorter Preparation

  1. SORTER AUTO TEST → TEST
  2. SORTER STOP START → STOP
  3. Nearest DCP_LSM → OFF, lock padlock
  4. CR TEST → ON (green LED on MCB solid)

Restore After Programming

  1. CR TEST → OFF
  2. DCP_LSM → ON, remove padlock
  3. SORTER AUTO TEST → AUTO
  4. SORTER STOP START → START

Programming New MCB — Auto Method

  1. Parameters tab → open saved parameter set
  2. Point IrDA dongle at MCB IR window (clean first)
  3. Click Auto
  4. Block write → Sorter Reset → CRC verify
  5. Success = ready | FAIL = retry 3×, then discard

CTB Test Screen — Use BEFORE Replacing MCB

💡
Point dongle at CTB, click Receive
0x00 constant = CTB not transmitting (check PLC O.Data bits 4-7). Valid speed code (e.g. 0x0F) = CTB works, problem is MCB IR window or CRB signal path. KB-12271: tape cardboard over MCB IR port, verify correct carrier gets comm error.
